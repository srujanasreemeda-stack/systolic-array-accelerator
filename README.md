# Systolic Array Accelerator

## Overview
Designed and implemented an FPGA-based systolic array accelerator for matrix multiplication using SystemVerilog and Xilinx Vivado.

## Features
- Parallel matrix multiplication
- Pipelined processing elements
- Parameterized RTL architecture
- FPGA implementation
- Timing analysis
- Functional simulation

## Architecture
The design uses interconnected processing elements (PEs) to perform parallel multiply-accumulate operations efficiently.

## Tools Used
- SystemVerilog
- Xilinx Vivado
- FPGA

## Verification
- Testbench verification
- Waveform debugging
- Functional simulation

## Results
- Improved computational throughput
- Efficient parallel processing
- Optimized FPGA resource utilization
